# Warehouse Audit Logic

This folder contains a few Python scripts I wrote to practice basic loops, lists, and dictionaries. I used a simple warehouse inventory theme to make the exercises more practical.

## Files
- `cycles_and_list.py`: The main script containing the logic.

## What it does
1. **Basic algorithms**: Calculating the total weight of a shipment and finding the heaviest box manually using `for` loops (without relying on Python's built-in `sum()` or `max()` functions).
2. **Filtering**: A function that takes a list of box weights and returns only the ones that fit within a specific range (10-30 kg).
3. **Dictionary practice**: A function that reads an inventory dictionary and prints alerts for items that are running low on stock.

## Goal
The main purpose of this code is to practice core Python fundamentals like control flow, variable scope, and manipulating data structures from scratch.
