# Python E-Commerce Data Manipulation 🛒

This repository contains Python practice scripts focused on core data structures and algorithms, specifically tailored for e-commerce scenarios. These exercises demonstrate clean, "Pythonic" approaches to common data handling tasks.

##  Topics Covered

* **List Comprehensions:** Efficiently filtering and transforming data in a single line (e.g., applying discounts to in-stock items).
* **Dictionary Operations:** Counting item frequencies using the optimized `.get()` method.
* **Custom Sorting (`lambda` functions):** Sorting lists of dictionaries based on specific keys (e.g., sorting products by price or users by rating, both ascending and descending).

##  Practical Scenarios Included

The code includes solutions to the following real-world tasks:
1. **Price Calculation:** Filtering a list of product dictionaries and applying a 10% discount only to items currently in stock.
2. **Category Counting:** Aggregating the occurrence of product categories from a user's shopping history.
3. **Search Results Sorting:** Ordering a list of products by price using custom `lambda` keys.
4. **Top Sellers Ranking:** Sorting a list of user ratings in descending order using the `reverse=True` parameter.
