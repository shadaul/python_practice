# Python Sandbox & Practice

This folder contains a sandbox script (`test.py`) where I test and practice various Python concepts. The script contains multiple commented-out code blocks that I use for learning.

## Concepts covered in this file:
1. **Control Flow & Logic**: A simple random number guessing game using `while True`, `if/elif/else`, and `break`.
2. **Object-Oriented Programming (OOP)**: Creating classes (`Mycar`, `Calculator`), understanding the `__init__` constructor, instance methods, and the `@staticmethod` decorator.
3. **Advanced Function Arguments**: Practicing `*args` (handling multiple positional arguments as tuples) and `**kwargs` (handling keyword arguments as dictionaries).
4. **List Iteration**: Unpacking nested lists and using the `enumerate()` function to track indexes.

## Goal
This is my personal playground for testing new syntax and understanding how different Python mechanisms work under the hood.
